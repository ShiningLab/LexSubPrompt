# PromptSub
This repository is for the paper Lexical Substitution as Causal Language Modeling.

## Dependencies
+ python >= 3.10.10
+ torch
+ wandb
+ lightning
+ transformers

## Run
Before training, please take a look at the **config.py** to ensure training configurations.
```
$ vim config.py
$ python main.py
```
